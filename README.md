# 🛠️ 週報後台管理系統 使用說明

本系統用於編輯與發佈教會週報，支援即時預覽、PDF輸出與前台同步。

---

## 🔐 1. 登入後台

1. 進入後台網址  
2. 輸入帳號（Email）與密碼  
3. 點擊「登入」  

---

## 🧭 2. 操作流程總覽

👉 建議每週操作流程如下：
基本資料 → 主題 → 信息大綱 → 詩歌 → 流程 → 小組 → 公告 → 預覽 → 儲存 / 發佈

---

## 🧩 3. 各功能說明

### ① 基本資訊
- 設定週報日期  
- 自動計算「第幾週」  
- 填寫年度異象  

### ② 信息主題
- 輸入主題標題  
- 設定講員、司會  
- 填寫經文  

### ③ 信息大綱
- 可新增「段落」與「要點」  
- 支援排序（上下移動）  
- 可刪除或調整內容  

### ④ 敬拜詩歌
- 分為「敬拜歌曲」與「回應詩歌」  
- 可新增 / 刪除 / 排序  

### ⑤ 聚會流程
- 設定時間（開始 / 結束）  
- 填寫流程內容  
- 可排序與新增  

### ⑥ 小組資訊
- 分為：
  - 平日小組  
  - 主日小組  
  - 特色 / 外縣市  
  - 海外分會  
- 可新增、編輯、排序  

### ⑦ 教會公告
- 可新增多則公告  
- 支援換行（自動變成條列）  
- 建議控制字數（避免版面溢出）  

---

## 👀 4. 預覽與版面檢查

- 可即時預覽 A4 版面  
- 系統會提示：  
  - ✅ 正常  
  - ⚠️ 內容過多（需精簡）  

👉 可切換：
- 「長篇大綱模式」

---

## 💾 5. 儲存資料

- 點擊「儲存」  
- 內容會同步至雲端資料庫  

---

## 📄 6. 匯出與發佈

### 匯出功能
- 匯出圖片（JPG / PNG）  
- 匯出 PDF（單頁 / 全部）  

### 發佈功能
- 一鍵上傳 PDF 至雲端  
- 自動同步前台網站  

---

## ⚠️ 使用注意事項

- 建議先完成所有內容再預覽  
- 若出現版面溢出：  
  - 減少文字  
  - 或開啟「長篇大綱模式」  
- 每次修改記得按「儲存」  

---

## 🚀 系統特色

- 即時預覽（所見即所得）  
- 自動排版 A4  
- 一鍵輸出 PDF  
- 雲端同步前台
